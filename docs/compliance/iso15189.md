# ISO 15189:2022 Mapping

HELIOS supports traceability, validation, and reproducibility evidence required by ISO 15189:2022.

## Coverage highlights

- **5.3 Equipment and software**: records runtime environment, containers, and signed outputs.
- **5.5 Examination procedures**: captures genome reference and transcript checks.
- **5.6 Assuring quality**: enforces container pinning and reproducible provenance.
- **7.3 Reporting results**: exports machine- and human-readable audit reports.
