# ISO 15189:2022 Mapping

HELIOS supports traceability, validation, and reproducibility evidence required by ISO 15189:2022 and is designed to produce comprehensive records of laboratory assay runs.

## Clause-level mapping

| ISO clause | HELIOS control | Evidence produced |
|---|---|---|
| 5.3 Examination processes | `reference_genome`, `container_pinning` | Reference source evidence, immutable container provenance |
| 7.3.2 Reporting of results | `vus_rate`, `mane_transcripts` | Variant interpretation metrics and transcript usage summaries |
| 7.7 Nonconforming work | Overall compliance scoring | Graded score and fail/warn breakdown across checks |
| 8.6 Risk management | `container_pinning` | Deterministic runtime package evidence and digest policy trace |

## AuditRecord and comprehensive run records

The immutable `AuditRecord` captures run identifiers, start/end times, input/output checksums, runtime container references, parameter sets, check outcomes, and cryptographic signatures. Together, these fields provide the comprehensive run records needed to demonstrate assay reproducibility, software control, and report traceability during ISO 15189 assessments.
