# GA4GH Standards Coverage

HELIOS aligns pipeline governance artifacts with core GA4GH standards.

## Standards touched

- **TRS**: container and pipeline reference traceability.
- **VRS / VRSATILE**: reference and variant annotation consistency checks.
- **Crypt4GH**: encrypted output presence validation.
- **RO-Crate / provenance interoperability**: export support for downstream FAIR tooling.
